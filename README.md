# NeonRealms - RPG Assincrono com Mestre IA

Plataforma web onde grupos de jogadores participam de campanhas de RPG de mesa assincronas, com um agente de IA atuando como Mestre de Jogo.

## Visao Geral

- **Jogo assincrono**: Jogadores respondem quando puderem
- **IA como Mestre**: Narrativa, combate e eventos gerenciados por IA
- **Sistema original**: Regras proprias (nao baseado em D&D)
- **Tema Cyberpunk**: Megacorps, implantes, hacks, estetica neon
- **Modular**: Arquitetura preparada para novos temas

## Status do Projeto

**Fase atual**: Planejamento

Veja as [Issues](https://github.com/9-realms-koder/prompts-and-dragons/issues) para acompanhar o progresso.

## Stack Tecnologico

| Camada | Tecnologia |
|--------|------------|
| Frontend | Next.js, React, TailwindCSS |
| Backend | Node.js/Bun |
| Banco de Dados | PostgreSQL + Prisma |
| Cache | Redis |
| Real-time | WebSocket/Socket.io |
| IA | Claude (Anthropic) |
| Infra | Docker, Turborepo |
